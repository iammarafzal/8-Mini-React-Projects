# Load More Data

A minimal React project that fetches and displays products with a "Load More" functionality.

## Features

- Fetches products from DummyJSON API
- Grid-based product display
- Load More button to fetch additional products
- Responsive design (4 columns on desktop, 2 on tablet, 1 on mobile)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
